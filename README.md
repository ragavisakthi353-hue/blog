# MERN Blog Platform

A full-stack blogging platform built with MongoDB, Express, React, and Node.js.

## Features
- User Registration and Login with JWT authentication
- Create, Read, Delete blog posts
- Commenting system on posts
- Modern, responsive UI with premium design

## Prerequisites
- Node.js installed
- MongoDB installed and running locally on default port `27017`

## Setup Instructions

1. **Start the Backend**
   Open a terminal, navigate to the `backend` directory, and run the server:
   ```bash
   cd backend
   npm run dev
   ```

2. **Start the Frontend**
   Open another terminal, navigate to the `frontend` directory, and run the app:
   ```bash
   cd frontend
   npm run dev
   ```

3. **Access the App**
   Open your browser and navigate to `http://localhost:5173`.
